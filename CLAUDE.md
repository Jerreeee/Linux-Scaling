## Repository Overview

This is a collection of independent system utility tools, each in its own subdirectory. Each tool is self-contained with no shared code or build dependencies between them. See the `CLAUDE.md` in each subfolder for details.

- **`Linux-Scaling/`** — Per-app UI scaling system for Linux (Bash scripts)
- **`KeyboardRemap/`** — Belgian AZERTY keyboard remapping (Windows + X11)
